# devuan-docker

Unofficial docker image for Devuan GNU/Linux

#### Supported arch and tags
- `amd64` `arm64` and `arm/v7` are supported
- `latest` tag is always the most recent version currently available. pull with:

      docker pull arfshl/devuan:latest

-  tags with release codename are also available, example: `daedalus`

       docker pull arfshl/devuan:daedalus
